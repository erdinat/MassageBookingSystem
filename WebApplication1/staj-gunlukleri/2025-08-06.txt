1. README.md:

```markdown
# L'OR Masaj Randevu Sistemi

Modern masaj merkezi randevu yönetim sistemi.

## Özellikler
- Müşteri randevu alma
- Terapist müsaitlik yönetimi
- Admin paneli
- Email notifications
- Payment simulation

## Kurulum
1. Backend: `dotnet run`
2. Frontend: `npm start`
3. Database: SQL Server

## Teknolojiler
- Backend: ASP.NET Core, C#, Entity Framework
- Frontend: React.js, Material-UI
- Database: SQL Server
- Authentication: JWT
```

2. Program.cs Configuration:
```csharp
var builder = WebApplication.CreateBuilder(args);

// DbContext servisini ekle
builder.Services.AddDbContext<ApplicationDbContext>(options =>
    options.UseSqlServer(builder.Configuration.GetConnectionString("DefaultConnection")));

// Notification Services ekle
builder.Services.AddScoped<IEmailService, EmailService>();
builder.Services.AddScoped<ISmsService, SmsService>();
builder.Services.AddScoped<INotificationService, NotificationService>();

// Auth Service ekle
builder.Services.AddScoped<IAuthService, AuthService>();

// Controller servislerini ekle
builder.Services.AddControllers();

// CORS ekle
builder.Services.AddCors(options =>
{
    options.AddPolicy("AllowAll", policy =>
    {
        policy.AllowAnyOrigin()
              .AllowAnyMethod()
              .AllowAnyHeader();
    });
});
```
